# Alice-Yandex_Tests

В этом тестовом задании 2 части:

Пройти кейсы, т. е выполнить определенную последовательность действий и посмотреть, совпадают ли фактические результаты действий с ожидаемыми. Запросы Алисе необходимо делать голосом.
Каждому кейсу нужно поставить вердикт: Pass – если фактические результаты по всем шагам соответствуют ожидаемым, либо Fail – если результат где-то отличается от ожидаемого; в таком случае нужно указать, в каком именно месте найдена ошибка.

Написать баг-репорт по любой из ошибок, найденных в предыдущей части задания.

Tests. xls - файл с тест-кейсами и баг-репортами

Папка Screens - вложения (скриншоты)
